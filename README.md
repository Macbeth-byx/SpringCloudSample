# Spring Cloud入门

## module描述

1. common-utils 公共工具
2. eureka-service 服务注册
3. api-zuul 服务网关
4. eureka-client 服务提供者（微服务）
5. eureka-consumer 服务消费者（微服务）

## 版本信息

1. java 1.8
2. spring-boot 2.2.0.RELEASE
3. spring-cloud Hoxton.RC1

## 已添加内容

1. eureka 注册中心
2. zuul 网关中心
3. ribbon 负载均衡，获取服务提供者地址
4. feign 声明式服务调用（简化ribbon）
5. Hystrix 服务容错保护

## 进行中
